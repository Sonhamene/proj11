# Exercícios de Vetores em JavaScript 🚀

Exercícios de lógica utilizando vetores, loops e condicionais em JavaScript.

---

# 📚 Conteúdos

- Arrays (Vetores)
- `for`
- `if`
- Média aritmética
- Vetores paralelos
- Interseção entre vetores

---

# 🧠 Exercício 1

Leia 7 nomes e mostre-os na ordem inversa.

## Conceitos

- Vetores
- Loop reverso
- Índices

## Exemplo

```js
Entrada:
Ana
Bruno
Carlos

Saída:
Carlos
Bruno
Ana
```

---

# 🧠 Exercício 2

Leia 8 números reais, calcule a média e mostre os valores acima dela.

## Conceitos

- Soma acumulativa
- Média
- Condicional `if`

## Fórmula

```text
média = soma / quantidade
```

---

# 🧠 Exercício 3

Leia nome e idade de 9 pessoas usando dois vetores relacionados. Mostre apenas os menores de idade.

## Conceitos

- Vetores paralelos
- Índices relacionados
- Filtro com `if`

## Exemplo

```js
nomes  = ["Ana", "Carlos"]
idades = [15, 22]
```

Saída:

```text
Ana tem 15 anos
```

---

# 🧠 Exercício 4

Leia dois vetores e crie um terceiro contendo os números que aparecem nos dois.

## Exemplo

```js
Vetor 1: [1, 2, 3, 4, 5]
Vetor 2: [3, 4, 5, 6, 7]

Resultado:
[3, 4, 5]
```

## Conceitos

- Comparação entre vetores
- `for` dentro de `for`
- Interseção
